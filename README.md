📜 Quote Generator

A modern Quote Generator web app that displays random quotes and allows users to share them on Twitter (X).

---

🚀 Live Demo

👉 https://mekihallil.github.io/quote-generator/

---

✨ Features

* 🎯 Generate random quotes
* 🔁 New quote button
* 🧠 Fallback to local quotes when API fails
* 🐦 Share quotes on Twitter (X)
* ⚡ Fast and responsive design

---

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API

---
📂 Folder Structure

```
quote-generator/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

---

🔗 API

https://dummyjson.com/quotes

> If the API fails, the app automatically uses local quotes.

---

🧠 How It Works

 Fetch quotes from API
 If API fails → use local quotes
 Display random quote
 Allow user to share via Twitter

---

🐦 Share Feature

Uses:

```
https://twitter.com/intent/tweet
```
